## 第 3 步：新增您的產品

### 方式 A：複製貼上（最簡單）

1. 找到最後一個產品（在 `]` 之前）
2. 複製下面的範本，貼在最後一個產品後面：

```
  {
    id: '新產品編號',
    name: '產品名稱',
    category: '分類',
    series: '系列名稱',
    price: 價格,
    description: '產品描述',
    benefits: ['功效1', '功效2'],
    size: '規格',
    volume: '容量'
  },
```

## 📋 各欄位說明

| 欄位 | 說明 | 範例 |
|------|------|------|
| `id` | 產品編號（不能重複） | `0-1` 或 `38-11` |
| `name` | 產品名稱 | `(0號)晶瑩剔透修膚霜(小)` |
| `category` | 分類代碼 | 見下方表格 |
| `series` | 系列名稱 | `微整形系列` |
| `price` | 價格（數字） | `2780` |
| `description` | 產品描述 | `晶瑩剔透，修護肌膚` |
| `benefits` | 功效標籤 | `['修護', '晶亮']` |
| `size` | 規格/尺寸 | `小` 或 `標準` |
| `volume` | 容量 | `30ml` 或 `50ml` |

**系統自動計算**：每 NT$100 = 1 點

- 價格 2780 = 27 點

## 🆘 常見問題

**Q: 我不小心刪除了什麼，怎麼辦？**
A: 按 `Ctrl+Z`（或 `Cmd+Z`）復原，或重新整理頁面。

**Q: 新增後看不到產品？**
A: 確認：
- 是否保存了文件
- `id` 是否與其他產品重複
- 系列名稱是否正確拼寫

**Q: 可以新增新的系列嗎？**
A: 可以！只要在 `series` 欄位填入新系列名稱即可。
